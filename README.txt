Тестовое задание - сокращатель ссылок "Shrtly"

Для работы приложения необходимо создать базу данных PostgreSQL с названием "shrtly" с дефолтным расположением
 (хост "localhost", порт "5432") и под правами пользователя "postgres" с паролем "postgres". 
Схему "public" восстановить из резервной копии "shrtly-25-08-2016.backup", расположенной в корне репозитория.

Для корректной работы приложении необходим его запуск по адресу localhost:8080

Незарегистрированным пользователям в приложении доступен только поиск - общее поле ввода поска тегов или информации о 
короткой ссылке (ссылка распознается по наличию в строке ввода подстроки "localhost:8080/s/").
